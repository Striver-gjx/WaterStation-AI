# 架构文档

存放系统架构图、技术选型说明、部署架构等。

## 技术栈

|------|------|
| Backend | Java 21 + SpringBoot 3 + MyBatis Plus |
| Admin Web | React + TypeScript + Tailwind CSS |
| Frontend | Vue3 + TypeScript + Element Plus |
| MiniProgram | Taro |
| Database | MySQL 8 + Redis |
| Storage | 本地文件系统（默认）/ OSS（可选） |
