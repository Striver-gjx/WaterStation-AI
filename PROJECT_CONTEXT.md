# Project Context

## 项目信息

- **项目名称**：WaterStation AI
- **行业**：桶装水配送
- **商业模式**：SaaS 平台
- **目标**：构建可交给 AI Agent 持续开发的工程

---

## 用户角色

| 角色 | 说明 |
|------|------|
| 消费者 | 通过小程序下单购水 |
| 门店 | 管理本店客户和配送 |
| 配送员 | 接单、配送、回收空桶 |
| 老板 | 查看经营数据、管理门店 |
| 总部 | 平台管理、数据分析 |

---

## 核心业务流程

```
客户下单 → 支付 → 派单 → 配送 → 空桶回收 → 库存变化 → 完成订单
```

---

## 核心对象

| 对象 | 说明 |
|------|------|
| Customer | 客户 |
| Product | 产品（水品类） |
| Order | 订单 |
| Driver | 配送员 |
| Inventory | 库存 |
| Bucket | 桶（押金管理） |
| Ticket | 水票 |
| Payment | 支付记录 |

---

## 系统架构

```
Customer (微信小程序)
    │
    ▼
Backend API (Java SpringBoot)
    │
    ▼
MySQL / Redis / OSS / MQ
```

---

## 部署目标

- Windows 端：一键安装部署
- macOS 端：一键安装部署
- 数据存储：本地文件（默认） / 外部数据库（可配置）
